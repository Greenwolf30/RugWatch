How to use the website · cloud · capacity · Actual Token Checker

Quick start

  1. Start the website from the RugWatch folder.
  2. Open http://127.0.0.1:8790/ (or your host and port).
  3. Add or upload wallets, then use Push cloud to save the list online.

Cloud list

  • Wallet list lives with the project on GitHub: data/wallets_cloud.json (and extra files when the list grows).
  • An index file tracks every cloud file: data/wallets_index.json.
  • About 100,000 wallets per cloud file and per local database file, then a new file is created automatically.

Local vs cloud

  • RugWatch Scan, Monitor, and the Wallets list use your local database.
  • Push cloud updates the online list that Actual Token Checker can read.
  • Actual Token Checker can flag holders from local and/or cloud lists, tagged [local], [cloud], or [both].

From Actual Token Checker

  • Open RugWatch from the yellow RugWatch link in Actual Token Checker.
  • Use the RugWatch checkbox on Analyze to include or skip flagged wallets.
  • On Ruggers, yellow Upload (Creator / Similar / Single) sends sellers into RugWatch and Push cloud.
  • Or Export a file and use Upload manual wallets here.
